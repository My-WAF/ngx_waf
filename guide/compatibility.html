# Compatibility Statement

# Platform Compatibility

This module does not provide compatibility support for Windows platforms.

# Nginx Compatibility

This module currently supports nginx-1.18.0 and nginx-1.19.7 versions.

NOTE

nginx-1.19.x is the mainline version, i.e. the development. Therefore, this module may not be compatible with the mainline version when it is updated. If you encounter compatibility errors when using the mainline version of nginx, you can raise issue (opens new window).

# Module Compatibility

# ngx_http_rewrite_module

This module is compatible with ngx_http_rewrite_module (opens new window) There is a compatibility issue. When the return or rewrite directives are used, the detection process of this module may be skipped.

See waf_mult_mount for the imperfect solution.
